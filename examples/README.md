# ตัวอย่าง

ไดเรกทอรีนี้มีตัวอย่างกรณีการใช้งานและการสาธิตของกรอบตัวแทน AI AI

## ตัวอย่างพื้นฐาน

- `simple_task.py`: แสดงให้เห็นถึงการใช้งานขั้นพื้นฐานของตัวแทน Anus AI สำหรับงานง่าย ๆ
- `web_search.py`: แสดงวิธีใช้เอเจนต์สำหรับการค้นหาเว็บและการดึงข้อมูล
- `code_generation.py`: ตัวอย่างของการใช้เอเจนต์สำหรับการสร้างรหัสและการดำเนินการ

## ตัวอย่างขั้นสูง

- `multi_agent_collaboration.py`: แสดงให้เห็นถึงความสามารถในการทำงานร่วมกันแบบหลายตัวแทน
- `document_processing.py`: แสดงวิธีการประมวลผลและวิเคราะห์เอกสาร
- `browser_automation.py`: ตัวอย่างของเบราว์เซอร์อัตโนมัติสำหรับงานเว็บ

## บทเรียน

-`Geting_started.md`: การสอนทีละขั้นตอนสำหรับการเริ่มต้นใช้งานกับทวารหนัก AI
- `custom_agent.md`: คู่มือสำหรับการสร้างบทบาทตัวแทนที่กำหนดเอง
- `tool_development.md`: การสอนสำหรับการพัฒนาเครื่องมือที่กำหนดเอง
